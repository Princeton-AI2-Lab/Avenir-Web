Abstract

Research Overview

Avenir-Web is an autonomous web agent framework designed for reliable execution of long-horizon tasks on complex, dynamic web interfaces. Addressing challenges in element grounding and long-term task tracking, it introduces a modular architecture combining Mixture of Grounding Experts (MoGE), Experience-Imitation Planning (EIP), and Adaptive Memory with Task-Tracking Checklists. This approach establishes a new open-source state-of-the-art on the Online-Mind2Web benchmark, bridging the performance gap with proprietary models in real-world deployments.

Method

Core Components

01

Mixture of Grounding Experts

Grounding combines screenshot understanding and browser state to produce actionable clicks, types, selections, and navigation steps.

02

Experience-Imitation Planning

Planning pushes the agent beyond one-step reactivity by generating higher-level task structure before detailed action selection.

03

Adaptive Memory

Memory stores evolving task context across long web trajectories, preserving intermediate findings and execution state.

04

Task-Tracking Checklists

Checklist-based tracking makes progress explicit, records partial completion, and keeps the agent aligned with remaining subgoals.

Execution Modes

Headless

Best for reproducible experiments, batch evaluation, and long-run execution.

Headed

Best when browser visibility matters but presentation overlays are unnecessary.

Demo Mode

Synchronized dashboard state, visible cursor feedback, and task progression designed for talks, videos, and analysis.

Quickstart

Installation and Execution

Install

conda create -n avenir-web python=3.11
conda activate avenir-web
pip install -e src
python -m playwright install chromium

API Key

export OPENROUTER_API_KEY="your-key"

Single Task

python example.py \
  --task "Find the official API docs for X" \
  --website "https://example.com/" \
  --mode headless

Batch Run

cd src
python run_agent.py -c config/batch_experiment.toml
